## Todo-App iTask
Detta är en Todo-applikation byggd med React (frontend), Express (backend) och Firestore som databas.

# Funktioner
- Skapa todos
- Uppdatera todos
- Radera todos
- Visa alla todos

---

## Starta Projektet

# 1. Klona projektet
```bash
git clone https://github.com/Amyholst1/grupprojekt
cd grupprojekt
```

----

# 2. Starta backend
Gå till backend mappen:

```bash
cd backend
npm install
node server.js
```

Backend körs på:
http://localhost:3002/

----

# 3. Starta frontend
Gå till frontend mappen:

```bash
cd frontend 
npm install
npm run dev
```

Frontend körs på:
http://localhost:5173/
